超详细spring-cloud教程|第二篇 服务消费者（Ribbon）

发表于 2017-06-21 | 分类于 spring-cloud | | 阅读

在上一篇文章，讲了服务的注册和发现。在服务架构中，业务都会被拆分成一个独立的服务，服务与服务的通讯是基于http restful的。spring cloud有两种调用方式，一种是ribbon+restTemplate，另一种是feign。在这一篇文章首先讲解下基于ribbon+rest。

Ribbon简介

ribbon是一个负载均衡客户端，可以很好的控制httP和tcp的一些行为。Feign也用到ribbon，当你使用@ FeignClient，ribbon自动被应用。
