WebService之CXF开发

发表于 2017-06-01 | 分类于 WebServcie | | 阅读
这是测试的

前言

Web service是一个平台独立的，低耦合的，自包含的、基于可编程的web的应用程序
从表面上看，Web service 就是一个应用程序，它向外界暴露出一个能够通过Web进行调用的API。这就是说，你能够用编程的方法通过Web来调用这个应用程序。我们把调用这个Web service 的应用程序叫做客户。
WebService解决的问题：
远程调用，跨平台调用，跨语言调用。

CXF和Axis2都是使用Java编写的WebService技术，不管是使用什么语言编写的webservice都要遵循SOAP协议（简单对象访问协议），

Webservice主要就是两部分组成：SOAP和XML代码片段。

这里DaMiss就说一下CXF的具体实现过程（服务端和客户端）
