超详细spring-cloud教程目录结构汇总

发表于 2017-06-20 | 分类于 spring-cloud | | 阅读

本教程所有项目都是基于Maven构建的，不熟悉Maven的可以去百度一下。

spring Cloud是基于springBoot的用于快速构建分布式系统的工具集，springBoot又是基于spring的，是快速开发spring项目的脚手架

本教程所有的项目都是用springboot的项目，快速构建一个spingboot项目：http://start.spring.io/ 里面可以选择构建工具，语言和springboot的版本。下面还有一些详细信息可以填。教程所有的项目用的是springboot的1.4.3release版本。

DaMiss保证这些项目都是经过测试成功之后才发布上去的，后续如果版本更新不适用的话，DaMiss会出博客更改的。

所有项目是在ecplise运行的。

项目目录

  • 超详细spring-cloud-教程|第一篇：服务的注册与发现（Eureka）

  • 超详细spring-cloud教程-第二篇-服务消费者（Ribbon）

  • 超详细spring-cloud教程-第三篇-服务消费者（Feign）

  • 超详细spring-cloud教程-第四篇-熔断器（Hystrix）

  • 超详细spring-cloud教程-第五篇-路由网关（Zuul）

  • 超详细spring-cloud教程-第六篇-高可用配置中心（Spring-cloud-config）

  • 持续更新中。。。。
